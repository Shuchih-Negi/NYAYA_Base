# ⚖️ NYAYA — AI-Guided Trauma-Informed Testimony System

> “The problem isn’t the survivor. It’s the system.”

NYAYA is an AI-assisted system designed to help survivors of trauma (sexual violence, abuse, trafficking) give testimony in a **safe, structured, and legally usable format** — without repeated re-traumatization.

---

## 🚨 Problem

Legal systems expect:
- Linear, consistent narratives  
- Immediate recall of events  

But trauma causes:
- Fragmented memory  
- Non-linear recall  
- Emotional distress during retelling  

As a result:
- Survivors are seen as “inconsistent”  
- Credibility is questioned  
- Testimony is repeated multiple times  
- Conviction rates remain low  

---

## 💡 Solution

NYAYA enables:

➡️ **One conversation → Complete legal evidence**

A **human-led, AI-assisted system** where:
- Survivors speak only once
- A trained counselor conducts the session
- AI works silently in the background

---

## 🧠 Core Features

### 🎤 1. Real-Time Testimony Capture
- Audio + Video recording
- Real-time transcription (multilingual)

### 🧩 2. Trauma Signal Analysis *(non-intrusive)*
- Voice patterns (pitch, pauses, energy)
- Facial distress indicators
- Outputs stress level (Calm → High Distress)

### 🧑‍⚕️ 3. AI Counselor Guidance
- Suggests trauma-sensitive questions
- Recommends when to pause or continue
- Flags topics to avoid
- Visible ONLY to counselor

### 🕰️ 4. Trauma Timeline Reconstruction ⭐
- Converts fragmented memory → event-based timeline
- Marks uncertainty (NOT treated as lies)
- Visualizes how trauma memory actually works

### 🔗 5. Entity Relationship Mapping
- Extracts people, places, objects
- Builds investigation-ready graph
- Helps link evidence

### ⚖️ 6. Legal Mapping (BNS)
- Maps testimony → relevant legal sections
- Uses semantic search (FAISS + embeddings)
- Explains laws in plain English

### 📄 7. Court-Ready Document Generation
- Structured testimony
- Timeline
- Legal sections
- Evidence references
- Exportable PDF
